# 前端重构检查清单

## 网络设置

- 运行 `npm run check:network`，确认小程序网络配置检查通过。
- 在小程序“我的 -> 网络设置”查看当前服务器地址。
- 真机调试时填写电脑局域网 IP，点击“测试连接”。
- 后台开发环境通过 `admin/.env` 设置 `VITE_DEV_API_TARGET`。
- 服务端通过 `CORS_ORIGINS` 管理允许访问接口的来源。

## 小程序界面

- 首页显示“敬一书院”“功能房预约”“可预约空间”，无乱码。
- 房间列表显示开放状态、容量、开放时间和审核要求。
- 房间详情页先展示空间信息，再进入时间线预约。
- 预约确认页必须填写手机号、一卡通号，并确认管理制度。
- 加载中、空数据、离线数据和网络失败都有明确提示。

## 后台管理端

- 侧边栏菜单和顶部标题显示正常中文。
- 工作台优先显示今日预约、待审核、使用中、今日异常。
- 待处理事项为空时显示空状态。
- 图表区域在窗口变化后能自适应。

## 验证命令

```bash
npm run check:network
cd admin && npm run build
cd ../server && npm run test:integration
```
