## 四、安全测试
| 测试项 | 预期结果 |
|--------|----------|
| HTTPS传输 | 所有API请求使用HTTPS |
| JWT认证 | 无token请求返回401 |
| CORS | 非白名单域名请求被拒绝 |
| SQL注入 | 输入特殊字符不导致SQL错误 |
| XSS | 输入脚本标签不执行 |
| 敏感数据 | 密码不返回前端 |
